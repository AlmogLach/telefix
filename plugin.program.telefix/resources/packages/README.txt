This folder contains zip files for ALL addons (71 bundles). Each zip has the addon folder as root (e.g. skin.bingie/addon.xml inside the zip).

If skin.bingie.zip is missing (e.g. Kodi was running when zips were built):
  1. Close Kodi completely.
  2. Run: build_all_addons.ps1 (in this folder) to create skin.bingie.zip, or manually zip the addons\skin.bingie folder and put skin.bingie.zip here.

Then run Telefix Wizard and choose "Install Telefix setup" to install all addons on this device.
